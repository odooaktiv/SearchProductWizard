Search Product

This module allows user to search matching products as per the attributes listed. Below listed are features accomplished in this module:

  • Product Attributes and Values is maintained with which the matching product will be displayed.
  • Products with its relevant attributes and values will be fetched.
  • A button with the Product count will also appear which will fetch all the relevant products.
  • A report can also be printed based on the search results of the products.

Prerequisites before Installation

There are no prerequisites required before the Installation

Configurations

Check the Attributes and Variants Boolean as true for availing the features of this module.

You can find the Attributes and Variants through:

Inventory > Configuration > Settings > Enable Product Variant.

User need to set attributes and then select the attributes as per their requirement. They can set the attributes in following simple steps:

Inventory > Configuration > Settings > Set Product Attributes

After setting the attributes you can further go to “Search Product” to find a product of your choice.

When you haven’t set the attributes, all products will be displayed. The attributes will be displayed only if you have set from the settings.
